## 🧠 Overview

**EmotionAI** is a next-generation **AI-powered Emotion Recognition Platform** designed for real-time analysis across **Text, Image, Webcam, Audio, and Dataset inputs**.

It leverages **Deep Learning (CNN), Machine Learning (SVM, RF), and NLP techniques** to detect, analyze, and visualize human emotions.

> 🔥 Built for **Tourism Safety, Smart Surveillance, and Behavioral Analytics**
